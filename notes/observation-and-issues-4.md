# User Observations, issues and new features required

1. The taxonomy studio language selection is not implemented as required. The expected behavior is:
 - It should be possible to select any of the listed languages, currently these are: En and Af. Other languages can be added in the future. 
 - If no language is selected, the language should default to UK English.
 - Selection of the language should cause only those taxonomy entries to be displayed that are in that language. For example, if Afikaans is selected, then only taxonoly items that are in the Afrikaans language, should be shown. Nested within the Afrikaans language item should be the flavors that are in the other languages. For example: Selecting Afirkkans and a taxonomy item 'Boerdery' should also optionally display the English taxonomy item 'Farming' nested within the 'Boerdery' item. This should be optional and the user should be able to toggle this feature on or off. Switching to English should cause only those taxonomy entries to be displayed that are in the English language by example, the item 'Farming' should have nested within it the flavors that are in the other languages, including 'Boerdery'. In this way the user will only need to manage the items of a single language instead of been inundated with items from all languages.
